## Software
There are **no expenses** for any of the software used while building and deploying the drones.
## Hardware
- **Base Station**: ~60€
- **Miscellaneous, Tools and Materials**: ~50€
- **The drone**: ~90€ without mapping sensors, ~170€ without mapping sensors  

## Total Estimated Cost
The total cost for building one drone with the Base Sation and Materials is about **~280€** with each additional drone consting **~170€**
