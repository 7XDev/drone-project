# Drones

The repository for the [ESP 32 Drones](https://www.alibaba.com/product-detail/ESP-Drone-Quadcopter-DIY-Educational-Learning_1600878049371.html) with software set up already and ready to go.

Find the official documentation [here](https://docs.espressif.com/projects/espressif-esp-drone/en/latest/gettingstarted.html#esp-drone-overview)

## Dependencies

This project is using a modified version of following repositories
 - [leeebo's cfclient](https://github.com/leeebo/crazyflie-clients-python)
 - [espressif's esp drone](https://github.com/espressif/esp-drone)
