# Page Layout 

This page describes the underlining layout of the web page.

## Website Overview

The website is envisioned to resemble the layout and functionality of [GitBook](https://homeverse.gitbook.io/home-verse-technical-document), offering a user-friendly and visually appealing interface.

### Webpages Structure

The website will feature a series of webpages that outline the various steps involved in constructing, programming, and operating drones, as well as using the associated software to design immersive drone shows.

#### Layout Design

- **Sidebar Navigation:**
  - A scrollable list on the left-hand side displaying the different steps, each represented by an icon and organized by category.
  - Categories will have the capability to include subcategories for better organization and ease of navigation.

- **Main Content Area:**
  - The central area will display detailed descriptions of each step, including instructions, diagrams, and other relevant resources.
  - At the bottom of each page, there will be navigation links to move to the next or previous step seamlessly.

- **Header Section:**
  - The header will feature a search bar, enabling users to search for specific terms, steps, or categories across the entire website efficiently.
